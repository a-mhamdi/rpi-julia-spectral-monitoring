
# Raspberry Pi & Julia

## Embedded DSP and Anomaly Detection

This repository documents the use of **Julia** for embedded signal processing and anomaly detection on the **Raspberry Pi**, covering the full communication stack required for real-world hardware interfacing:

- Digital `I/O`
- `I2C`
- `UART` serial
- `SPI`

The **Raspberry Pi** is a capable single-board computer well-suited for edge computing, industrial monitoring, and embedded systems development. **Julia**, a high-performance language designed for numerical and scientific computing, brings the expressiveness of high-level code with execution speeds comparable to compiled languages — making it a strong choice for on-device signal processing workloads.

Together, they form a practical platform for implementing **FFT-based spectral analysis** and **real-time anomaly detection** directly at the edge, without offloading computation to a remote server.
